# Real Estate Market & Affordability Dashboard

## Project Overview
A data-driven analysis of US real estate affordability — tracking price-to-income 
ratios, median home values, and market trends to identify where housing is 
becoming unaffordable and where opportunity still exists.

## Business Questions Answered
1. Which metro areas have the worst price-to-income affordability ratios?
2. How have home values changed relative to income over the past 5 years?
3. Which markets are overvalued vs undervalued relative to local incomes?
4. What does the affordability landscape look like across US regions?

## Tools Used
- **Python** — pandas, matplotlib, data cleaning and analysis
- **Tableau Public** — interactive affordability dashboard (live link)
- **Dataset** — Zillow Research Data + US Census ACS income data (public)
